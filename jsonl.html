O que é JSONL?

      
JSONL (JSON Lines) é um formato onde cada linha é um JSON completo. É comum em logs e exportações. Esta página converte cada linha em um bloco JSON formatado, separando por linhas em branco.

      
    
        
  • Posso colar um JSON grande? Sim, mas JSONL é “um por linha”. Para JSON comum, use o formatador.
    • 
        
  • Ele junta tudo em um array? Não; ele só embeleza por linhas para leitura.
