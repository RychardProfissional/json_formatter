Quando usar escape/unescape

      
Escape ajuda quando você precisa inserir texto (com quebras, aspas, tabs) dentro de uma string JSON. Unescape reverte sequências como \n, \t e \" para leitura normal.

      
    
        
  • Isso cria um JSON completo? Não; ele transforma texto para caber em uma string JSON.
    • 
        
  • O que acontece com caracteres especiais? Eles são escapados de acordo com JSON.
