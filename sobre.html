Sobre

Ferramentas simples para trabalhar com JSON no dia a dia.

Objetivo

Este site reúne utilitários para formatar, validar e transformar JSON de forma rápida, direto no navegador. A ideia é reduzir tempo de depuração e facilitar a leitura/compartilhamento de payloads de API, configurações e logs.

Privacidade e processamento

As transformações de JSON são feitas localmente (client-side). Isso significa que o conteúdo que você cola ou carrega é processado no seu dispositivo. Para entender como cookies, anúncios e medição funcionam, consulte a Política de Privacidade.

Monetização e transparência

Para manter o projeto no ar, podemos exibir anúncios. Os espaços são reservados no layout para reduzir impacto visual (CLS) e o carregamento pode depender do seu consentimento, conforme explicado nas preferências de cookies.
