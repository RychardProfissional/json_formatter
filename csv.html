JSON → CSV

Converta JSON (objeto ou array de objetos) para CSV.

Pronto
Dica: o modo CSV → JSON aceita CSV simples (sem parsing avançado de aspas). Para arquivos complexos, ajuste o CSV antes. 


          

            
          

        

      

    


    

      
O que este conversor suporta

      
    
        
  • JSON → CSV: objeto ou array de objetos, com colunas unificadas por chaves.
    • 
        
  • CSV → JSON: CSV simples com header na primeira linha.
    • 
      

      
    
        
  • Arrays/objetos aninhados viram o quê? Eles são serializados como JSON dentro da célula.
    • 
        
  • Por que meu CSV com aspas não converte perfeito? Este é um parser simples; para casos avançados, prefira uma conversão dedicada.
